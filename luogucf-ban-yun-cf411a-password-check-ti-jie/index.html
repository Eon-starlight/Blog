Luogu（CF搬运）CF411A 【Password Check】【题解】

2020-03-30

题目链接

  • 介绍一种题解里没有的方法突然在脑里生成的奇怪方法

  • 这需要用到字符串的强大find功能
  • 加入a是一个字符串，a.find()（这里填你需要寻找的字符或字符串）
  • 接着如果找到了就返回位置（最靠前的下标，从0开始）
  • 反之没找到就返回-1
  • 举例：
string a="123";
int x=a.find('1'),y=a.find("23"),z=a.find('4');
- 这时x的值为0，y为1，z为-1
- 接着我们需要用字符型变量做循环变量
- 就向这样：

```cpp
for(char i='1';i<='9';i++)
  • 接下来就用介绍的功能完成这道题
  • 首先设计个变量：
string a;
bool x,y,z;
  • a为读入变量，x判断是否有数字，y判断是否有小写字母，z判断是否有大写字母。
  • 接着读入后，写三个循环分别查看是否有数字，小大写字母：
for(char i='0';i<='9';i++)//判断数字
    {
		if(a.find(i)!=-1)//发现就存起来
		{
			x=1;
			break;
		}
	}
	for(char i='a';i<='z';i++)//判断小写字母
    {
		if(a.find(i)!=-1)//同上
		{
			y=1;
			break;
		}
	}
	for(char i='A';i<='Z';i++)//大写字母
    {
		if(a.find(i)!=-1)//同
		{
			z=1;
			break;
		}
	}
  • 最后只要看xyz再输出就可以了
  • 整体代码：
#include<cstdio>
#include<iostream>
#include<string>
#include<cmath>
#include<algorithm>
#include<map>
#include<set>
#include<queue>
#include<deque>
using namespace std;
string a;//设置变量
bool x,y,z;
int main()
{
	cin>>a;//读入（只能cin）
    for(char i='0';i<='9';i++)//解释过的三个循环
    {
		if(a.find(i)!=-1)
		{
			x=1;
			break;
		}
	}
	for(char i='a';i<='z';i++)
    {
		if(a.find(i)!=-1)
		{
			y=1;
			break;
		}
	}
	for(char i='A';i<='Z';i++)
    {
		if(a.find(i)!=-1)
		{
			z=1;
			break;
		}
	}
	if(x&&y&&z&&a.length()>=5/*字符串长度函数*/)printf("Correct");//如果条件成立
	else printf("Too weak");//否则
    return 0;
}
